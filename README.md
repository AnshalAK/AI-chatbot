# AI Chatbot for Conversational Applications

## Overview
This project is a web-based AI chatbot designed to handle basic conversational queries using open-source natural language processing models. It provides real-time responses through a simple and interactive user interface.

The goal of this project is to demonstrate the integration of frontend and backend technologies with conversational AI models in a full-stack application.

---

## Key Features
- Real-time conversational interaction  
- Web-based chatbot interface  
- Integration with open-source AI models  
- Modular frontend and backend architecture  
- REST API communication  

---

## Tech Stack

### Frontend
- React  
- HTML, CSS, JavaScript  

### Backend
- Flask (Python)  
- REST APIs  

### AI / NLP
- Hugging Face Transformers  
- Pre-trained conversational models (DialoGPT / BlenderBot)  

### Tools
- Git & GitHub  
- Postman  

---

## System Architecture
1. User input is captured in the React frontend  
2. The message is sent to the Flask backend via an API request  
3. The backend processes the input using a pre-trained conversational AI model  
4. The generated response is returned to the frontend in real time
